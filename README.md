# RSC解密系列

从零开始，搭建一个支持RSC和Server Actions的应用。不使用任何现有的框架，也不使用任何打包工具，来帮助大家更好的理解RSC和Server Actions。

## System Requirements

- [git][git] v2.18 or greater
- [NodeJS][node] v18 or greater
- [npm][npm] v8 or greater
